# Нелинейное интегральное уравнение

- #π/определение:
	- **Нелинейное интегральное уравнение** — интегральное уравнение, в котором искомая функция не может быть выделена из ядра.
- #π/формула:
	- $\displaystyle y(x)=\int_{a}^{b} K(x,t,y(t)) \, dt + f(x)$.
